# Welcome to Amajon

[Amajon](https://amajon.herokuapp.com/#/) is a Amazon clone with products and departments that specifically appeal to Korean culture.

![Introduction](https://media.giphy.com/media/QmFvEQ5RzxiZSI7oF0/giphy.gif)

# Technologies Used

* Ruby on Rails
* React/Redux
* Javascript
* AJAX
* JBuilder
* PostgreSQL
* HTML5
* CSS
* AWS S3
* ASW IAM
* Heroku
